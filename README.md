# Quiz
A C++ program which simulates a quiz<br>
Uses basic data structures and data files<br>
This quiz has two menus: **admin** and **player**
### Admin
Admin can enter, modify, delete and display the questions
### Player
The player has a choice whether to give a science, gk or history quiz<br>
He can choose either a **basic mode** or a **pro mode**

### Marking scheme:
20 qs<br>
**Basic** : +4 / -1 / 0<br>
**Pro**   : +5 / -2 / -1<br>
Pro will be time based, each question will be visible for 10sec
